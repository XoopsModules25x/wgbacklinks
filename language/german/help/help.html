Help: wgBacklinks

Description

The wgBacklinks module can be used in CMS XOOPS

Install/uninstall

No special measures necessary, follow the standard installation process and extract the wgbacklinks folder into the ../modules directory. Install the module through Admin -> System Module -> Modules.

Detailed instructions on installing modules are available in the XOOPS Operations Manual

Features

The module wgBacklinks can be used to admin your backlinks of serveral websites from one central provider website
